FoundationLab

Web app

Overview

FoundationLab, a software development agency, hired me as UX and Visual Designer to help build a tool to improve the way their remote team reviewed leads and calculated a project's potential profitability. I worked alongside a Project Manager and a Developer. I was responsible for information architecture, wireframes, and visual design.

Product Architecture

It's easy to say Yes to every good feature idea. The problem with that is you'll eventually find yourself with a bloated product that has users resenting their employers for making them learn how to use it. I use sitemap diagrams to get everyone on the same page. This particularly important when working with an entirely remote team.
