FoundationLab

Web app

Overview

FoundationLab is a software development agency. They hired me to help build a tool to improve the way their remote team reviews client leads and calculate a project's potential profitability. I worked alongside a Project Manager and a Developer and was responsible for information architecture, wireframes, and visual design.

The Problem

FoundationLab uses an online form to solict applications from potential clients. The FoundationLab teams receives these forms as emails. This makes it difficult to assign and track leads without constant back-and-forth emailing.

Product Architecture

There were a lot of good feature ideas for this product, but saying yes to all of them would have created a monster. I use sitemap diagrams to get everyone on the same page. Clear documentation is particularly important for a completely remote team.
